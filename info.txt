	UI
Форма 0:
0.1 - кнопка "Загрузить данные" - выполняет метод 1
0.2 - кнопка "Изменить данные" - открывает форму с предыдущей версии(4)
0.3 - кнопка "Перейти к первому этапу" - скрывает эту форму, открывает форму 1
0.4 - кнопка "Перейти ко второму этапу" - скрывает эту форму, открывает форму 2
0.5 - кнопка "Выход" - завершение работы программы
Форма 1:
1.1 - таблица входных данных - размещения информации о поставщиках, заказчиках и расходов на единицу товара (читает с класса данных ТранспДанные)
1.2 - рабочая таблица - распределение перевозок, отображение точек ss-метода (читает данные с класса АльтернативноеРешение)
1.3 - кнопка "С-З угол" - определяет начальное распределение перевозок (начальное АР1), делает доступной кнопку 1.4; становится недоступной после нажатия
1.4 - кнопка "Шаг" - делает шаг в текущей итерации; когда шаги заканчиваются, делает доступной кнопку 1.7, 
1.6 - список сигм - добавляется при каждом нажатии кнопка 1.4
1.7 - кнопка "Выбор" - при выделенном пункте списка 1.6 переходит на следующую итерацию (начальное АР1 = выбранному); если чекбокс 1.10 - истина, то вызывает метод 
1.8 - кнопка назад - закрывает окно, сохраняя состояние в класс формы, возобновляет форму 0
1.9 - метка "Z=" - показывает последнее значение Z, управляется методом 2
1.10 - чекбокс "Определить оптимальное решение автоматически" - при значении ИСТИНА вызывает метод определениеМинимальнойСигмы()
Форма 2:
2.1 - таблица текущего распределения растрат и +-; при инициализации формы таблица соответствует значению начального АР2
2.2 - таблица следующего распределения растрат и +-
2.3 - метка "Z=" - показывает последнее значение Z для нечёткого числа, управляется методом 3
2.4 - кнопка "Шаг" - делает шаг по текущей итерации (таблица 2.2 в таблицу 2.1, новое значение таблицы 2.2 пересчитывается)
2.5 - список значений Z - пополняется при каждом нажатии на кнопку 2.4
2.6 - список эпсилон - пополняется при каждом нажати на кнопку 2.4
2.7 - кнопка "Выбор" - если выбран элемент из списка 2.5, то скрывается эта форма и открывается форма 3; если выбран элемент списка 2.6, то переходит на следующую итерацию (начальное АР2 = выбранному)
Форма 3:
3.1 - таблица решений - колонки: отображённые значения Z(3), поиск лучшего АР по всем критериям(MM,BL,S,HW,HL,G,P)
3.2 - кнопка "Выбор" - переходит на следующую итерацию(начальное АР2 = выбранному), закрывает при этом эту форму и возобновляет форму 2
